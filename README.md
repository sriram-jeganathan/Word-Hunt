# 🎯 Word Hunt (C++ Console Game)

Welcome to **Word Hunt**, a fast-paced word guessing game inspired by the classic Hangman. Built using modern C++, this project focuses on clean design, interactive gameplay, and efficient use of core programming concepts.

---

## 🎮 Gameplay Overview
- A hidden word is randomly selected  
- You guess one letter at a time  
- Correct guesses reveal the word  
- Wrong guesses bring you closer to defeat  
- Win by uncovering the full word before running out of attempts  

---

## ✨ Highlights
- 🎲 Random word generation for replayability  
- 🧩 Smart input handling and validation  
- 📊 Tracks progress, guesses, and remaining attempts  
- ⚡ Responsive and interactive terminal experience  
- 🏁 Clear win/lose outcomes
